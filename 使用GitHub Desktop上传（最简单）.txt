              使用 GitHub Desktop 上传项目（最简单的方法！）
================================================================================

如果命令行推送失败，GitHub Desktop 是最简单的解决方案！

🎯 为什么推荐 GitHub Desktop？
-----------------------------
✅ 完全图形化界面，无需命令行
✅ 自动处理身份验证
✅ 一键推送，不会出错
✅ 适合新手使用

📋 详细步骤
----------

第1步：下载并安装 GitHub Desktop
------------------------------
1. 访问：https://desktop.github.com/
2. 点击 "Download for Windows"
3. 运行下载的安装程序
4. 安装完成后打开 GitHub Desktop

第2步：登录你的 GitHub 账号
--------------------------
1. 在 GitHub Desktop 中，点击 "File" → "Options"
2. 点击 "Accounts" 标签
3. 点击 "Sign in to GitHub.com"
4. 在浏览器中登录你的 GitHub 账号
5. 授权 GitHub Desktop
6. 完成登录

第3步：添加本地项目
------------------
1. 在 GitHub Desktop 中，点击 "File" → "Add local repository"
2. 点击 "Choose..." 按钮
3. 浏览到项目目录：
   C:\Users\Administrator\Desktop\shuchu\android_app
4. 点击 "选择文件夹"
5. 如果提示"这不是Git仓库"，点击 "create a repository" 

第4步：发布到 GitHub
-------------------
1. 在 GitHub Desktop 左下角，确认有文件变更
2. 在底部的 "Summary" 框中输入：Initial commit
3. 点击蓝色的 "Commit to main" 按钮
4. 点击顶部的 "Publish repository" 按钮
5. 在弹出窗口中：
   - Name: kuaishou-video-replacer-android
   - Description: 快手视频替换工具 Android版
   - 取消勾选 "Keep this code private"（必须是公开仓库）
6. 点击 "Publish repository"
7. 等待上传完成

第5步：查看自动构建
------------------
1. 上传完成后，访问：
   https://github.com/lijunjing159/kuaishou-video-replacer-android
2. 点击 "Actions" 标签页
3. 你会看到正在构建的任务
4. 等待15-25分钟
5. 构建完成后，点击任务
6. 在页面底部 "Artifacts" 区域下载APK

❓ 常见问题
----------

Q: GitHub Desktop 是免费的吗？
A: 是的，完全免费！

Q: 需要配置什么吗？
A: 不需要！只需要登录GitHub账号即可

Q: 上传后多久能得到APK？
A: 15-25分钟，GitHub Actions会自动构建

Q: 怎么下载APK？
A: 访问仓库 → Actions → 点击构建任务 → 底部Artifacts下载

Q: 以后修改代码怎么办？
A: 在GitHub Desktop中：
   1. 修改代码后会自动检测到变更
   2. 写提交信息
   3. 点击"Commit to main"
   4. 点击顶部的"Push origin"
   5. GitHub会自动重新构建APK
